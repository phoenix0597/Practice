### Итоги модуля 5
1. Для форматирования строк и подстановки значений используется метод format()
   Также, при необходимости, выводом этих значений можно управлять с помощью плейсхолдеров.
   path = 'C:\{user}\temp\file.txt'.format(user=user_name) 
   print("{:.2f} рублей".format(price))
2. Для разбиения строки по разделителю используется метод split()
   names_lst = input("Введите имена через запятую: ").split(",")
3. Для объединения списка в строку используется метод join()
   people_str = "&".join(people)
4. Если нужно проверить начало или конец строки на нужные нам символы, можно использовать методы
   startswith() и endswith(), которые возвращают True/False:
   path.startswith('C:\') # True/False
   path.endswith('.txt') # True/False
5. Строку можно привести к единому регистру - верхнему или нижнему:
   word = input().upper()  # приводит все символы строки к верхнему регистру
   word = input().lower()  # приводит все символы строки к нижнему регистру

### Шпаргалка по методам строк:
Метод              Назначение
#### Проверки (возвращают True или False):
S.isdigit()        Состоит ли строка из цифр
S.isalpha()        Состоит ли строка из букв
S.isalnum()        Состоит ли строка из цифр или букв
S.islower()        Состоит ли строка из символов в нижнем регистре
S.isupper()        Состоит ли строка из символов в верхнем регистре
S.startswith(str)  Начинается ли строка S с шаблона str
S.endswith(str)    Заканчивается ли строка S шаблоном str
#### Методы, которые возвращают новую строку:
S.split(символ)    Разбиение строки по разделителю
S.join(список)     Сборка строки из списка с разделителем S
S.upper()          Преобразование строки к верхнему регистру
S.lower()          Преобразование строки к нижнему регистру
S.title()          Первую букву каждого слова переводит в верхний регистр, а все остальные в нижний
S.format()         Форматирование строки
#### Методы, которые возвращают число
S.find(str)        Поиск подстроки в строке. Возвращает номер 1-го вхождения. Если таковых нет, то возвращает (-1)
S.rfind(str)       Такой же поиск, но с конца строки. Возвращает номер 1-го вхождения. Если таковых нет, то возвращает (-1)   
S.index(str)       Поиск подстроки в строке. Возвращает номер первого вхождения Если таковых нет, то вызывает ValueError
S.count(str)       Возвращает количество непересекающихся вхождений подстроки
